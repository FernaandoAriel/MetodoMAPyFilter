Ejemplos de map y filter en JavaScript

Los métodos map() y filter() son funciones de orden superior en JavaScript que permiten manipular arrays de manera elegante y funcional. Estos métodos son fundamentales para la programación funcional.

¿Qué es map()?

El método map() crea un nuevo array con los resultados de llamar a una función proporcionada en cada elemento del array original.

const nuevoArray = array.map((elemento, índice, array) => { /* código */ });

Resultados de map:

Ejemplo 1: Duplicar números
let numeros = [1, 2, 3, 4, 5];
let duplicados = numeros.map(function(num) {
    return num * 2;
});

Ejemplo 2: Convertir a mayúsculas
let nombres = ["juan", "ana", "carlos"];
let nombresMayusculas = nombres.map(function(nombre) {
    return nombre.toUpperCase();
});

¿Qué es filter()?

El método filter() crea un nuevo array con todos los elementos que cumplan la condición implementada por la función proporcionada.

const nuevoArray = array.filter((elemento, índice, array) => { /* código que retorna true o false */ });

Resultados de filter:

Ejemplo 1: Filtrar números pares
let numerosPares = numeros.filter(function(num) {
    return num % 2 === 0;
});

Ejemplo 2: Filtrar mayores de edad
let personas = [
    { nombre: "Juan", edad: 25 },
    { nombre: "Ana", edad: 17 },
    { nombre: "Carlos", edad: 30 }
];
let mayoresDeEdad = personas.filter(function(persona) {
    return persona.edad >= 18;
});

Comparación entre map() y filter()

Característica map() filter()
Propósito principal Transformar elementos Seleccionar elementos
Tamaño del array resultante Igual al original Igual o menor al original
Valor de retorno de callback Nuevo valor para el array Booleano (true/false)
Uso común Convertir datos Buscar/filtrar datos

Combinando map() y filter()

Estos métodos son aún más poderosos cuando se combinan:

// Duplicar solo los números pares
let numerosParesDuplicados = numeros
    .filter(num => num % 2 === 0)
    .map(num => num * 2);

document.getElementById("combined-result").textContent = "Números pares duplicados: " + numerosParesDuplicados.join(", ");
